# msh-MOLED
msh-MOLED for multi-shot multiple overlapping-echo detechment imaging.

----

DESCRIPTION

ep_seg_se_oled.dll & ep_seg_se_oled.so: msh-MOLED pulse sequence executable on 3T whole-body scanners (Prisma, Siemens Healthcare, Germany).

MID*.mat: complex-valued k-space data of msh-MOLED (input of network model).
